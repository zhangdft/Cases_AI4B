# AI4Bat 课程案例库 (Cases_AI4B)

本仓库包含了“人工智能在电池研发中的应用 (AI4Bat)”课程的所有实践案例，旨在引导学生了解利用 AI 相关技术加速电池研发效率的概念。

## 案例目录

### [Case 1: 机器学习入门 —— 薪资预测实战 (Salary Prediction)](Case_1/)
- **内容**：通过一个简单的 ML 模型训练，介绍数据探索、特征工程、模型对比及解释。
- **目标**：建立机器学习的基本工作流概念。

### [Case 2: 材料研发的三大范式 —— 以电池抗枝晶材料设计为例](Case_2/)
- **内容**：以剪切模量预测为案例，对比实验试错、计算指引 (DFT) 和 ML 辅助三种范式。
- **重点**：
  - **DFT 计算**：以 LiF 为例的第一性原理计算流程。
  - **ML 优化**：多模型对比、特征去冗余、高通量筛选思路。
- **目标**：理解 AI 如何与传统物理计算结合，加速材料筛选。

---

## 环境要求
建议使用 `conda` 创建名为 `qimeng` 的环境，并安装以下主要依赖：
- `numpy`, `pandas`, `matplotlib`, `seaborn`
- `scikit-learn`, `xgboost`
- `matminer` (用于 Case 2)
